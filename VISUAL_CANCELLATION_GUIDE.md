## ❌ Common Issues and Solutions

### Issue 1: "Cannot Cancel" Message
```
┌─────────────────────────────────────────────┐
│ ❌ Cannot Cancel Booking                    │
│                                             │
│ Cancellation not allowed. Bookings can     │
│ only be cancelled at least 3 days before   │
│ the flight.                                │
└─────────────────────────────────────────────┘
```
**Solution:** Your flight is less than 3 days away. Cancellation is not possible.

---

### Issue 2: "Authentication Required"
```
┌─────────────────────────────────────────────┐
│ ⚠️ Please Login                             │
│                                             │
│ You need to be logged in to cancel         │
│ bookings.                                  │
└─────────────────────────────────────────────┘
```
**Solution:** Go back to Step 1 and login first.

---

### Issue 3: Button Not Working
**What to Check:**
1. Open browser console (Press F12)
2. Look for error messages
3. Common errors:
   - "Token found: No" → Login again
   - "Failed to fetch" → Backend not running
   - "Network error" → Check internet connection

**Solution:**
1. Make sure backend is running: `cd backend && npm start`
2. Make sure frontend is running: `npm run dev`
3. Clear browser cache and try again
4. Try a different browser

## 🎯 Quick Checklist

Before cancelling, make sure:
- [ ] You are logged in
- [ ] Backend is running (port 5000)
- [ ] Frontend is running (port 5175)
- [ ] Flight is at least 3 days away
- [ ] Booking status is "Confirmed"
- [ ] You have internet connection

## ✅ Summary

**To cancel your ticket:**
1. Login → My Tickets
2. Find booking → Click "Cancel Booking"
3. Select reason → Continue
4. Choose refund method → Continue
5. Accept terms → Confirm Cancellation
6. Done! Check email for confirmation

**Remember:** You can only cancel if flight is 3+ days away!
