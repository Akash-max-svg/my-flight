# 🎯 QUICK CANCELLATION REFERENCE CARD

## ⚡ FASTEST WAY TO CANCEL

```
Login → My Tickets → Find Booking → Cancel Ticket → Confirm → Done!
```

---

## 📋 3 SIMPLE STEPS

### Step 1: Navigate
```
Click "🎫 My Tickets" in navigation bar
```

### Step 2: Cancel
```
Find your booking → Click "❌ Cancel Ticket"
```

### Step 3: Confirm
```
Select reason → Choose refund method → Confirm
```

**That's it!** ✅

---

## ⏰ IMPORTANT RULES

### ✅ CAN CANCEL IF:
- Flight is **>3 days** (72 hours) away
- Booking is **not already cancelled**
- Flight has **not departed**

### ❌ CANNOT CANCEL IF:
- Flight is **<3 days** away
- Booking is **already cancelled**
- Flight has **already departed**
