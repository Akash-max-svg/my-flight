## Step 3: Update Your .env File

After creating/resetting the user, update your connection string:

### If you used autogenerated password:

Open `backend/.env` and update:

```env
MONGODB_URI=mongodb+srv://akashraj:YOUR_NEW_PASSWORD@cluster0.ko7quug.mongodb.net/?appName=Cluster0
```

Replace `YOUR_NEW_PASSWORD` with the password you copied.

### If you kept password as "akashraj":

Your current connection string should work:
```env
MONGODB_URI=mongodb+srv://akashraj:akashraj@cluster0.ko7quug.mongodb.net/?appName=Cluster0
```

### ⚠️ Important: Special Characters in Password

If your password contains special characters, you need to URL encode them:

| Character | Encoded |
|-----------|---------|
| @ | %40 |
| : | %3A |
| / | %2F |
| ? | %3F |
| # | %23 |
| [ | %5B |
| ] | %5D |
| ! | %21 |

**Example**:
- Password: `Pass@123!`
- Encoded: `Pass%40123%21`
- Connection: `mongodb+srv://akashraj:Pass%40123%21@cluster0...`
