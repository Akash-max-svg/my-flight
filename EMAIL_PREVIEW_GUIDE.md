## 📥 TICKET DOWNLOAD PROCESS

### Step-by-Step:

```
1. User receives email
   ↓
2. Opens email in inbox
   ↓
3. Sees "Download E-Ticket" button
   ↓
4. Clicks button
   ↓
5. Browser opens download URL
   ↓
6. Backend generates PDF
   ↓
7. PDF downloads automatically
   ↓
8. User has ticket!
```

### Download URL Structure:

```
http://localhost:5000/api/bookings/{bookingId}/ticket
                      ↑                ↑
                   Backend          Booking ID
```
